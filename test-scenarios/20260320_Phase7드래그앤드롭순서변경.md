# 테스트 시나리오 — 드래그 앤 드롭 단계 순서 변경

- **생성일**: 2026-03-20
- **대상 Phase**: Phase 7 (추가 기능)
- **구현 기능**: LV3 단계별 상세에서 @dnd-kit 기반 드래그 앤 드롭으로 StepCard 순서를 변경하면 colIndex가 자동 재부여되고, LinearFlow·SwimLane 다이어그램에 즉시 반영

**사전조건**: `가공비 마감` 프로세스 LV3 진입 (스텝 3개)

**실행 단계**:
1. 단계별 상세 섹션에서 각 StepCard 왼쪽 확인

### TC-002 [정상] 👤 수동 — 스텝 순서 교환 후 카드·흐름도 동시 갱신

**사전조건**: `가공비 마감` LV3 진입 (① 가공비 현황 조회 → ② 전표 생성 → ③ 생성 결과 검증)

**실행 단계**:
1. ③ 생성 결과 검증의 `⠿`를 잡고 ① 가공비 현황 조회 위로 드래그
2. 드래그 중 원래 카드가 반투명이 되고 미리보기 카드가 마우스를 따라다니는지 확인
3. 마우스를 놓음
4. 상단 "전체 흐름도" LinearFlow 순서 확인

**입력값**: 없음 (드래그 동작)

**예상 결과**:
- 단계 목록 순서: ① 생성 결과 검증 → ② 가공비 현황 조회 → ③ 전표 생성
- 번호 뱃지가 새 순서대로 1, 2, 3으로 갱신됨
- 전체 흐름도 노드 순서가 카드 순서와 동일하게 변경됨

**확인 방법**: StepCard 순서 ↔ LinearFlow 노드 순서가 일치하는지 비교
