추가 6. 새 컴포넌트 (섹션 7 하단에 추가)
-- 7.8 AddMethodModal --
props: { onClose, onSelectDirect, onSelectAI }
용도: 프로세스 추가 방법 선택
구현:
  - 두 가지 카드 형태 선택지
  - ✍️ 직접 입력: onSelectDirect 호출 → 기존 AddModal
  - ✨ AI 자동 생성: onSelectAI 호출 → AIGenerateModal

-- 7.9 AIGenerateModal --
props: { deptId, groupId, dept, onClose, onComplete }
용도: AI 프로세스 자동 구조화 3단계 위저드
구현:
  Step 1 — 입력 폼:
    - 프로세스명 (필수)
    - 담당 부서 (기본값: 현재 그룹 부서)
    - 담당자
    - 업무 흐름 설명 (textarea, A→B→C 또는 자유 텍스트)
    - [✨ AI로 구조화] 버튼

  Step 2 — 로딩:
    - "AI가 프로세스를 분석 중입니다" 메시지
    - 프로그레스 바 애니메이션
    - Claude API 호출 (claude-sonnet-4-20250514)

  Step 3 — 결과 미리보기:
    - 생성된 단계 목록 (읽기 전용 카드)
    - 각 카드: 단계명 / 화면명 / PT / Logic 요약
    - [← 다시 입력] [편집기에서 열기 →]
    - "편집기에서 열기" 클릭 시 onComplete(process) 호출

  API 프롬프트:
    "다음 업무 흐름을 분석하여 JSON 배열만 응답하세요.
     배열 외 다른 텍스트 없이 JSON만 출력하세요.
     업무흐름: {userInput} / 담당부서: {dept}
     각 항목: title, screenName, dept, pt, logic, warning"

  에러 처리:
    - API 실패 → "다시 시도" 버튼 표시
    - JSON 파싱 실패 → Step 1로 복귀, 안내 메시지
    - API 키 없음 → 직접 입력 방식으로 안내
